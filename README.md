# Music-Project

## Requirements for this project:

- [X] JSON processing
- [x] Pandas 
- [ ] Search Engine components (Ranking, Crawling)
- [X] Web scraping
- [x] Word Cloud
- [x] Matplotlib | plots & charts
- [x] Python libraries

# Instructions:
## How to install all requirements:
Type this into cmd:
```
pip install -r requirements.txt
```

## How to run our project:
Type this into cmd:
```
python src/main.py
```
